HTML Semantics

The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages.

Semantics Elements

Tag Name Description
<section> Section defines a section in a document
<article> Article specifies independent, self-contained content.
<header> Header represents a container for introductory content or a set of navigational links.
<footer> Footer defines a footer for a document or section
<nav> Nav defines a set of navigation links
<aside> Aside defines some content aside from the content it is placed in (like a sidebar).
<figure> Figure pecifies self-contained content, like illustrations, diagrams, photos, code listings, etc.
<figcaption> Figcaption defines a caption for a <figure> element
< main> Main Specifies the main content of a document
<time> Time Defines a date/time
